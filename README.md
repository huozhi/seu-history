# 东南校史知识竞赛初赛系统

### 技术栈

Backend: Django
Frontend: Bootstrap, jQuery

### 介绍

每个人利用自己的学号和一卡通号登陆

每个人题目随机，但是刷新和重新登陆保证题目相同

计算得分，限时保证不可以超时，超时自动提交答案
